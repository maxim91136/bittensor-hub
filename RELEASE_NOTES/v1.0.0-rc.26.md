# Release v1.0.0-rc.26

**Release Date**: 2025-12-08

## Introduction of Subnet Ranking System

### New Features
- **Position Change Indicators**: Visual ranking changes with ▲ (up), ▼ (down), NEW (new entry)
- **History Comparison**: Fetches previous snapshot to calculate position changes
- **Combined Rank Column**: Rank number + change indicator in single column (e.g., "1 ▲2")

### Improvements
- **Unified Monospace Fonts**: Consistent `Courier New` across entire subnet table
- **4-Column Layout**: Rank | Subnet | Share | Daily (streamlined from 5)
- **Market Conditions Title**: Smaller on desktop, larger on mobile for better readability

### Technical Details
- Fetches `/api/top_subnets_history?limit=2` for previous ranking data
- Green ▲ for rank improvements, Red ▼ for drops, Blue NEW for new entries
- Responsive column widths: 15% | 35% | 25% | 25%
