# Release v1.0.0-rc.26.3

**Release Date**: 2025-12-08

## Last Updated Timestamps in Tooltips

### New Features
- **Global Last Updated**: All key tooltips now display "Last updated" timestamp
- **Immediate Display**: Timestamps appear on page load, not just after refresh

### Updated Tooltips
- **Block Height**: Shows block number + last updated
- **Block Time**: Shows calculation details + last updated
- **Staking APY**: Shows APR breakdown + last updated
- **Price Change Pill**: Shows 1h/24h/7d changes + last updated
- **Halving Pill**: Shows projection details + last updated
- **TAO Price Badge**: Shows price + last updated

### Technical Details
- `window._lastUpdated` now set in both `initDashboard()` and `refreshDashboard()`
- Data sourced from Taostats API (`last_updated` or `_timestamp` field)
- Timestamps formatted with `toLocaleString()` for user's locale

### Bug Fixes
- Fixed: Tooltips not showing Last Updated on initial page load
- Fixed: Price Badge tooltip using undefined variables
