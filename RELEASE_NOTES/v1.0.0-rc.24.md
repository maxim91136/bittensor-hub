# v1.0.0-rc.24 (2025-12-07)

Major dashboard enhancement with new Market Conditions card, layout reorganization, and Fear & Greed Index integration.

### Added

- **Market Conditions Card**: New comprehensive market analysis card featuring:
  - **Market Signal**: Visual indicator (Bullish/Bearish/Caution/Watch) with color-coded badges
  - **Volume & Price Metrics**: 24h volume change and price change with +/- indicators
  - **Market Phase Analysis**: Contextual phase detection based on multiple factors
  - **Moving Averages Grid**: Real-time MA tracking (2h, 4h, 3d, 7d) with % change from current price
  - **Confidence Metrics**: Data quality indicators (sample count, hours of data)
  - **Matrix Console Style**: Monospace font, green glow effects, responsive grid layout
  - **Light/Dark Mode Support**: Adaptive styling for both themes

- **Fear & Greed Index Integration**:
  - Integrated into market signal calculation for enhanced sentiment analysis
  - Combines volume trends, price action, and market sentiment
  - Provides contextual market phase descriptions

- **Last Updated Timestamps**: All tooltips and cards now show when data was last refreshed
  - Consistent formatting across all elements
  - Better transparency for data freshness

### Changed

- **Card Layout Reorganization**:
  - Swapped API Status card (now position 7) with Neurons card (now position 11)
  - Market Conditions card positioned at top of dashboard for maximum visibility
  - Improved visual hierarchy and user workflow

- **API Status Card Redesign**:
  - Centered badge (Error/OK) within card for better visual balance
  - Responsive positioning (55% vertical on mobile, 50% on desktop)
  - Maintained icon and label in standard positions
  - Absolute positioning with transform for pixel-perfect centering

- **Typography & Spacing**:
  - Market Conditions title reduced to 1.3em on desktop (was 1.6em)
  - Card spacing standardized with --gap-lg variable (12-16px)
  - Consistent gap values across all breakpoints

- **F&G Card Adjustments**:
  - Spoon gauge moved lower (margin-top: 50-65px)
  - Better vertical balance on large screens
  - Improved visual hierarchy

### Improved

- **Tooltip System**:
  - English translations for all tooltips and code comments
  - Improved market phase clarity and descriptions
  - Pills stay open on hover (no auto-hide)
  - Desktop info badges persistent (X-button removed for better UX)

- **Market Intelligence**:
  - Market phase awareness in Ampelsystem (volume signal logic)
  - Multi-factor analysis combining volume, price, MAs, and sentiment
  - Enhanced confidence scoring based on data quality

### Technical

- **New Files**:
  - `market-conditions.css`: Matrix-style card styling with responsive grid
  - `market-conditions.js`: Market data processing and card updates

- **Modified Files**:
  - `index.html`: Card reorganization, Market Conditions card HTML structure
  - `style.css`: API Status centering, spacing variables, F&G adjustments
  - `script.js`: Fear & Greed integration, tooltip enhancements

- **CSS Architecture**:
  - Used CSS Grid for metrics layout (4 columns desktop, 2 mobile, 1 small)
  - Absolute positioning with transform for precise badge centering
  - Flexbox with align-items: stretch for full-height containers
  - clamp() for responsive typography

- **API Integration**:
  - Fetches Fear & Greed Index from external API
  - Combines with Taostats aggregates and volume history
  - Fallback handling for API failures
