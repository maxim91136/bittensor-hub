Release: v1.0.0-rc.21

Fear & Greed UI polish, Ampelsystem (Volume signal) robustness, and image optimization.

Highlights
- Improved Fear & Greed card layout with history matrix and spoon-shaped gauge. Supports user-provided spoon graphics (WebP preferred).
- Ampelsystem tuning: tradedShare guard, MA alignment, hysteresis, and debug-friendly runtime overrides via `window.applyVolumeConfig()`.
- Converted spoon PNGs to optimized WebP and updated loader to prefer WebP.
