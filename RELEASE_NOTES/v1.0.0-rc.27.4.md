# v1.0.0-rc.27.4 - Bittensor SDK v10.0 Compatibility

**Release Date:** 2025-12-10

## What's Fixed

### Bittensor SDK v10.0 Breaking Changes

The Bittensor SDK v10.0 was released on December 10, 2025 with breaking API changes that caused our dashboard to display zeros for Subnets, Validators, and Neurons.

This release fully migrates to the new SDK v10.0 API:

| Old API (v9.x) | New API (v10.0) |
|----------------|-----------------|
| `bt.subtensor()` | `bt.Subtensor()` |
| `subtensor.get_subnets()` | `subtensor.get_all_subnets_netuid()` |
| `bt.Metagraph(netuid=n, network=net)` | `subtensor.metagraph(netuid=n, mechid=0)` |

## Technical Details

- Updated `fetch_network.py` for SDK v10.0 compatibility
- Updated `fetch_top_subnets.py` for SDK v10.0 compatibility
- All GitHub Actions workflows now pass successfully
