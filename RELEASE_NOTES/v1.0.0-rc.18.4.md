# Release v1.0.0-rc.18.4

**Release Date:** December 3, 2025

## Summary

This release unifies the Volume Signal tooltip system and adds a neutral glow state to the Ampelsystem for better user discoverability.

## Changes

### Volume Signal Tooltip Unification

- **Single Source of Truth**: Tooltip content is now managed exclusively in `script.js`
- **Removed duplicate code** from `taostats-card.js` that was causing tooltip flickering
- **Combined content**: Ampelsystem signal + confidence + Moving Averages all in one tooltip
- **Removed colored emoji** from confidence line to reduce visual noise

### Ampelsystem Improvements

- **Added neutral signal emoji**: `âšª Neutral` now shows in tooltip for consistency
- **Added white glow for neutral**: Subtle white pulse animation indicates the card is active
- **Removed conflicting border styles**: `taostats-card.css` no longer overrides Ampelsystem colors

### Bug Fixes

- **Fixed red border on neutral signal**: CSS `.stat-card.pulse-down` rules were overriding Ampelsystem
- **Fixed tooltip inconsistency**: No more flickering between old MA-based and new Ampelsystem tooltips
