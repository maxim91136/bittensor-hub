# v1.0.0-rc.28.1 - Real-Time Binance Pricing & TAO vs BTC Chart

**Release Date:** 2025-12-10

## Highlights

### Real-Time Price Data from Binance
- **< 1 second delay** vs 20+ minutes from previous source (Taostats)
- Price, 24h change, and volume all from Binance ticker
- Fallback chain: Binance → Taostats → CoinGecko

### TAO vs BTC Performance Comparison
- **New toggle** in price chart to compare TAO and BTC side-by-side
- See how TAO performs relative to Bitcoin over any time range
- Normalized percentage view for easy comparison

### EUR Currency Support
- **Toggle USD/EUR** on both price pill and chart
- Live EUR/USD conversion rate from Binance
- Synced and persisted to localStorage

### Max Price Range (~600 Days)
- View complete TAO price history since Binance listing (April 2024)
- Smart date labels: "Apr '24" format for long time ranges
- 12 evenly spaced labels for clean display

## What's New in RC28

### Added
- Binance API as primary price source (real-time)
- TAO vs BTC comparison toggle on price chart
- EUR currency toggle on price pill and chart
- Max time range for ~600 days of history
- Daily chunking for all history data types

### Performance
- Workflow intervals reduced from 15min to 5min
- Fear & Greed API calls reduced to 2x daily

### Fixed
- Max chart x-axis labels now properly formatted
- Market Conditions uses Binance 24h change (consistent with pill)
- Mobile responsive improvements for chart toggles and tables

### Changed
- Binance is now primary price source, Taostats is fallback
- Disclaimer updated to reflect "Price Data" from Binance
